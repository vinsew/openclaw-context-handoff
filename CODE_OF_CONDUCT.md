# Code of Conduct

This project aims to be calm, practical, and respectful.

## Expected behavior

- Be respectful and constructive.
- Focus criticism on ideas and implementations, not people.
- Assume good intent while still being honest about problems.
- Prefer clear examples and reproducible reports over vague complaints.

## Unacceptable behavior

- Harassment or intimidation
- Personal attacks
- Doxxing or sharing private information
- Repeated bad-faith disruption

## Maintainer responsibility

Maintainers may remove or reject contributions, issues, or comments that make
the project unsafe or unproductive.

## Reporting conduct issues

Please report serious conduct issues privately to the repository maintainer.
